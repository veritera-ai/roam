# Code of Conduct

ROAM is infrastructure for autonomous agent organizations. Discussion here should stay technical, specific, and useful.

## Expected Behavior

- Be respectful and direct.
- Assume good intent, but stay precise.
- Keep issues focused on reproducible problems, documentation gaps, SDK/protocol questions, or public integration work.
- Protect private information: do not post credentials, `.roam/` state, agent logs, private prompts, customer data, proprietary artifacts, or confidential PRD material.
- Report vulnerabilities privately to `security@veritera.ai`.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Posting private data or secrets.
- Publicly disclosing security vulnerabilities before coordinated disclosure.
- Spam, low-effort promotion, or unrelated debate.
- Attempts to pressure maintainers into exposing protected commercial internals through public issues.

## Enforcement

Maintainers may edit, hide, lock, or remove issues, pull requests, discussions, or comments that violate this code of conduct. Repeated or serious violations may result in a ban from the project.

For moderation concerns, contact Veritera through `security@veritera.ai` and include enough context for review.
