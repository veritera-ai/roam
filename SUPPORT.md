# Support

Use the public issue tracker for:

- install problems
- reproducible public-component bugs
- documentation gaps
- SDK or protocol questions
- framework adapter questions

Do not post secrets, `.roam/` session state, agent logs, credentials, customer data, private prompts, or proprietary artifacts.

## Security

Send vulnerabilities to:

```text
security@veritera.ai
```

Do not open public security issues.
